## Getting started 


### Clone Repo
[https://github.com/Rotweinpirat257/webapp.Redlist]

### Clone repo

```bash

git clone {https://github.com/Rotweinpirat257/webapp.Redlist]
```
### Create venv

```bash

python -m venv venv 
source venv/bin/activate


```

### Install requierements

```bash

pip install -r requirements.txt


```

### Initialized Database 

In Terminal
```bash
Flask init-db

```


### Run Redlist

```bash

flask run

```
